##  介绍

基于 `botpy` 开发的 QQ娱乐机器人，目前仅支持 `windows` 平台。

### 功能介绍

<details>
<summary>功能列表</summary>

#### 基本功能
- [x] 签到：获得金币
- [x] 查询：查询玩家数据

### 实用工具
- [x] 60s：每天60秒读懂世界
- [x] 一言：输出随机语录

</details>

## 使用

1. 运行 `Modernia.exe` 。
2. 配置 `config\config.yml` 文件。
3. 运行 `start.bat` 文件。
